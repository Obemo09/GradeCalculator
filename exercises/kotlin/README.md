# Kotlin Collection Exercises

This folder contains solutions to the Kotlin collection exercises given in class.

## Exercises Included

### Exercise 1 – Higher Order Function
Created a function `processList` that takes:
- A list of integers
- A lambda predicate

The function returns a new list containing only elements that satisfy the predicate.

Example:
Input:
[1,2,3,4,5,6]

Output:
[2,4,6]

---

### Exercise 2 – Transforming Between Collection Types
Given a list of words, a map was created where:
- Key = word
- Value = length of the word

Then entries with length greater than 4 were printed.

Example Output:
apple has length 5  
banana has length 6  
elephant has length 7

---

### Exercise 3 – Complex Data Processing
Using a `Person` data class, the program:
1. Filters people whose names start with A or B
2. Extracts their ages
3. Calculates the average age
4. Prints the result rounded to one decimal place

---
